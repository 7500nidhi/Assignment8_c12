# Assignment 8: Supervised Learning Classification

## Project Title
Titanic Survival Prediction

## Description
This project uses the built-in Titanic dataset from the Seaborn library to predict passenger survival using supervised machine learning classification.

## Dataset
- Titanic Dataset (Seaborn)

## Models Used
- Logistic Regression
- Decision Tree

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
- Classification Report

## Results
- Logistic Regression Accuracy: 77.42%
- Decision Tree Accuracy: 70.97%

Logistic Regression performed better and was selected as the best model.

## Libraries Used
- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
